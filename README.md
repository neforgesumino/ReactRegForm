# React Sign up Form \ Форма регистрации на React
For Eng go below

Просто форма регистрации с валидацией. Присутствует статичный роутер, две страницы - главная и страница ошибки. Также доступна смена дизайна (с чёрного на красный). Использовались хуки.

A simple React registration form with validation, I used custom hooks and static react router. There are two pages - main page and 404. You can toggle the design. Thank you for viewing.
